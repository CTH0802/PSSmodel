\section{Build up a model}
We build a model that can reproduce the observed streamer by considering a self-collapse density enhancement structure with a rotation velocity and evolve with time in 3D space. 
\section{Formalism} \label{Sub-sec: Formalism}
In the formalism, including the density enhancement along the polar angle ($\theta$), infall velocity ($v_{\rm inf}$), rotation velocity ($v_{\rm rot}$), accumulated rotation ($\phi$), and inclination (i).
The sonic length.
\begin{align}
    r_{\rm s} = c_{\rm s} T
\end{align}
Infall velocity
\begin{align}
    \displaystyle {v_{\rm inf}=-\sqrt{\frac{2GM_*}{r}}-3.3c_s}~~(c_s=200 m/s)
\end{align}
The rotation velocity.
\begin{align}
    \displaystyle v_{\rm rot} = r_{\rm s}\omega\Omega_{\rm kep}\left[\left(\frac{r}{r_{\rm s}}\right)^{-1} + \left(\frac{r}{r_{\rm s}}\right)^{-\frac{1}{3}}\right]
\end{align}
The accumulated rotation.
\begin{align}
    \displaystyle \phi = \phi_0 + t_{\rm s} \omega\Omega_{\rm kep}\sqrt{\frac{2c_{\rm s}^3t_{\rm s}}{GM_*}}\left[\left(\frac{r}{r_{\rm s}}\right)^{-\frac{1}{2}} + \left(\frac{r}{r_{\rm s}}\right)^{-\frac{1}{3}}\right]
\end{align}

    \end{array} \right.
\end{align}
Rotation velocity
\begin{align}
    \left\{ \begin{array}{l}
         v_x = -v_{\rm rot}\sin{\theta} \sin{\phi} + v_{x_{\rm inf}}\\
         v_y = v_{\rm rot}\sin{\theta} \cos{\phi} + v_{y_{\rm inf}}\\
         v_z = 0 + v_{z_{\rm inf}}\\
    \end{array}
    \right.
\end{align}
\section{Streamer on the sky in the PPV space}
Position rotates by inclination.
Inclination(i): backward (+), forward (-). 
\begin{align}
    \left\{ \begin{array}{l}
     x' = r\sin{\theta} \cos{\phi} \\
     y' = y\cos{i}+z\sin{i} \\
     z' = -y\sin{i}+z\cos{i}=r(-\sin{\theta} \sin{\phi} \sin{i}+\cos{\theta} \cos{i}) \\
    \end{array} \right.
\end{align}


Velocity rotates by inclination
\begin{align}
    \left\{ \begin{array}{l}
         v_x' = v_x\\
         v_y' = v_y\cos{i}+v_z\sin{i}=v_{rot}\sin{\theta} \cos{\phi} \cos{i}+v_{\rm inf}(\sin{\theta} \sin{\phi} \cos{i}+\cos{\theta} \sin{i})\\
         v_z' = -v_y \sin i + v_z \cos i = - v_{\rm rot}\sin{\theta}\cos{\phi}\sin i + v_{\rm inf}(-\sin{\theta} \sin{\phi}\sin i + \cos{\theta}\cos i)\\
    \end{array}
    \right.
\end{align}

$\theta$ corresponds to the initial polar angle of the streamer, $\phi$ is the initial azimuth angle, $i$ is the inclination angle (zero inclination corresponds to an edge-on disk), $\omega$ is the ratio of Keplerian rotation velocity, and $t_{\rm s}$ is the collapsing time of the particle within the streamer.

\subsection{Global fitting of the full model}
We model the streamer kinematics using a five–dimensional parameter space ($\Theta_0$, $\Phi_0$, i, $t_{\rm s}$, $\omega$), corresponding to the initial polar and azimuthal angles of the streamline, the inclination of the system, the characteristic collapse timescale, and the ratio of rotation velocity. For each target, we first perform a coarse 5D grid search to identify reasonable regions of parameter space. The grid covers $\Theta_0 \in (0, \pi/2)$, $\Phi_0 \in (0, 2\pi)$, and $i \in (-\pi/2, \pi/2)$ with uniform sampling, while the collapse time $t_{\rm s}$ is sampled logarithmically around a characteristic dynamical timescale set by the Keplerian period at a reference radius, and $\omega$ is sampled linearly between 0 and 1.

The grid search uses the same 11 streamer centroids extracted from the data to evaluate the model performance. At each grid point, we compute the misfit between the model streamline and the observed streamer centroids using the error metric $E = \sqrt{\left\langle \Delta x^2 + \Delta z^2 + w_v\,\Delta v^2 \right\rangle}$, where $\Delta x$ and $\Delta z$ are the differences in projected position (in AU), and $\Delta v$ is the corresponding line-of-sight velocity residual (in $\rm km s^{-1}$) for each of the measured centroids. The weight $w_v = (\Delta v_{\rm km\,s^{-1}} / \Delta x_{\rm AU})^2$ is chosen such that spatial and velocity deviations contribute in comparable physical units.

We then use the grid results to construct the priors for the subsequent MCMC exploration. Specifically, we select all grid models with $E \le (1 + f)\,E_{\min}$, where $E_{\min}$ is the minimum grid misfit and f is typically 0.05$\sim$0.10 (relaxed if necessary to include at least several tens of models). For each parameter we take the 5th$\sim$95th percentile range of these “good” models and pad this interval by $\sim30\%$ to define a broad, uniform prior. This procedure yields data-driven prior bounds that are wide enough to avoid artificially constraining the solution but narrow enough to exclude clearly disfavored regions of parameter space.

Starting from these priors, we run an ensemble Markov Chain Monte Carlo (MCMC) sampler (emcee) in the same 5D space. For the fast likelihood based on the 11 streamer centroids, we use of order $N_{\rm walkers}\sim 20$ and $N_{\rm steps}\sim 1.5\times10^4$, initializing the walkers around the best grid solution but clipping them to the prior ranges.
